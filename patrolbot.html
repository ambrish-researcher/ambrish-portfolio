PatrolBot

Autonomous Waypoint Navigation Robot

Overview

PatrolBot is an autonomous navigation system developed using Python and Object-Oriented Programming. The robot follows predefined waypoints, detects obstacles, and performs real-time path adjustment.

Environment Input
